Installation Instructions:
1. In your WHMCS root folder, upload the contents of the upload directory. You should end up with 2 files in these paths (relative to WHMCS root folder):
	modules/gateways/payiota.php
	modules/gateways/callback/payiota.php

2. Log in to your WHMCS admin panel and go to Setup, Payments, Payment Gateways.

3. In the Activate Module dropdown, select PayIOTA.me and click the Activate button.
		
4. In the PayIOTA.me configuration form enter your API Key and your Verification key (from your PayIOTA.me account.)

5. Make sure your System URL (WHMCS admin settings) is set correctly, relating to whether your site uses HTTP or HTTPS.
